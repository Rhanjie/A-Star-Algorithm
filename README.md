 ### Configuration CMake
 Create file **./cmake-modules/Config.cmake** and set your SFML ROOT directory inside, eg.
 ```cmake
 set(SFML_ROOT ../SFML-2.5.0/)
 ```
After this click File -> Reload CMake Project in CLion
